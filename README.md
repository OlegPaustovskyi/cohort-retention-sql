# User Retention & Cohort Analysis Project

This project focuses on evaluating user retention metrics (Retention Rate) based on raw historical activity logs. It consists of two main parts: SQL data transformation and an interactive dashboard.

## 🛠️ Tech Stack
- **SQL (PostgreSQL)**: Data cleaning, CTEs, filtering, and cohort calculation.
- **Google Sheets**: Pivot tables, conditional formatting, and interactive slicers.

## 📊 Deliverables
1. **SQL Script**: You can find the fully optimized data-cleaning script directly in this repository (`cohort_analysis.sql`).
2. **Interactive Dashboard**: View the full visualization, triangular cohort tables, and retention analysis in Google Sheets here: [**👉https://docs.google.com/spreadsheets/d/1TX8HqibUWR5pN-FSRwrqCHmnk0dZ2R6Ius1BMBgfMP8/edit?usp=sharing**]
